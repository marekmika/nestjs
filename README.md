## Description

Starter-repo - Nest with Prisma and PostgreSQL

## Prerequisites
Create and fill up .env file in root folder based on .env-dev file 

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn dev
```

## Test
Be aware that execution of this command will drop and load fixtures

```bash
# e2e tests
$ yarn test:e2e
```

## Generate 
Generate sections of app

```bash
# plop
$ yarn plop:generate
```
